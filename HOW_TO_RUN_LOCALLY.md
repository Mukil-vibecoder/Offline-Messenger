# How to Open & Run WITHOUT Appylics

Since you have the complete source code, you can run this project directly on your computer if you install **Android Studio**.

## Option 1: View Code Only (Simplest)
You are already doing this! You can open any file in this folder using your current text editor (VS Code, Notepad++, etc.) to read the code.
- **Logic**: Go to `app/src/main/java/com/example/emergencymessenger`
- **Design**: Go to `app/src/main/res/layout`

## Option 2: Run the App (Requires Android Studio)
To build the APK or run it on a phone directly from your computer:

1.  **Download Android Studio**:
    -   Go to [developer.android.com/studio](https://developer.android.com/studio) and install it.
2.  **Open Project**:
    -   Launch Android Studio.
    -   Select **Open**.
    -   Navigate to `c:\Science Exhibition\OfflineMessenger` and click OK.
3.  **Sync**:
    -   Wait for Gradle to sync (download dependencies). It make take 5-10 minutes the first time.
4.  **Run**:
    -   Connect your Android phone via USB.
    -   Enable **USB Debugging** on your phone (Settings -> Developer Options).
    -   Click the green **Run (Play)** button in Android Studio.

This will install the app directly onto your phone without using any website.
